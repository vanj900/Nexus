## How to Use These Examples

Each example is a completed `WORLD_OVERVIEW.md` fill-in. Use them to:

1. **Study the process** — See how the questions connect to each other across the steps.
2. **Steal shamelessly** — Adapt any of these worlds for your game, campaign, or story.
3. **Seed your own loops** — Use the JSON seeds to generate variations procedurally.
4. **Identify patterns** — Notice how the expert review notes at the end of each example caught real problems that needed fixing.

## Contributing Examples

If you build a world using Nexus and want to share it, see [CONTRIBUTING.md](../CONTRIBUTING.md). Examples are one of the most valuable contributions you can make — they demonstrate what the framework produces in practice.

Good examples to add:
- **Space opera** — interstellar empires, alien contact, generation ships
- **Urban fantasy** — contemporary cities with hidden magical systems
- **Solarpunk** — genuinely functional utopias with their own tensions
- **Historical** — ancient civilisations, specific historical periods
- **Weird fiction** — cosmic horror, surreal settings, deliberately broken physics
